WEEK 1

Welcome to the A Cloud Guru Accelerator Program for AZ-104 Microsoft Azure Administrator. Congratulations on taking the first steps towards preparing for your certification! We understand it can feel a little overwhelming to study for an important exam – don’t worry, we’ve got you covered.

You’re about to embark on a twelve week journey to build the skills and confidence that you need to pass your exam with flying colors! Here’s how this will work:

This program uses a Watch, Do, and Read framework – each week, you’ll receive a lesson plan from us, including which lessons to watch, which labs and assessments to do, and which resources to read. For optimal success, we recommend investing 3-4 hours per week on this program. Want a bird’s eye view of the content? Check out this course overview.

This exam follows the Microsoft Job Role assessment model to ensure that a Microsoft Certified: Azure Administrator Associate has the required knowledge and skills to handle the Azure Administrator job.

Let’s get started, shall we? We’ll begin by reviewing the exam structure so you know what to expect when the time comes! Then we’re off to explore the services used to manage Azure, from the friendly web interface, to the robust command line tools, and the powerful world of Azure Resource Manager (ARM) Templates.
