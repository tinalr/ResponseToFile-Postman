Watch Now: Migrating 17 years of infrastructure to Azure

Technical debt has a way of sneaking up on you. And when you’ve been building websites as long as Microsoft Partner Program Manager Scott Hanselman, you’ll end up with quite a few interesting challenges when it’s time to move all that infrastructure to the cloud.

In this on-demand webinar, Scott will walk us through the key considerations of migrating apps to Azure - whether it’s Hanselman.com or your own business:

  • Continuous integration and deployment with GitHub and Azure DevOps
  • Provisioning separate dev/staging/production environments with Azure App Services and Deployment Slots
  • Updating a legacy .NET codebase for the cloud
  • Removing single points of failure and managing URLs with Azure Front Door
  • CDNs, storage, and more!
